# CORO

CORO is an ADHD-friendly cyber pet productivity assistant that acts as a virtual body double.

## Planned Features

- Pixel-art cyber pet
- To-do list integration
- Focus session support
- Break reminders
- Water and self-care reminders
- "Let's Work" button
- "Later" button with pet reactions

🚧 Currently in development (v1)
